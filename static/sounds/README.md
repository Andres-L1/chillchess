# WHITE NOISE - SONIDOS FALTANTES

Los archivos de sonido no están incluidos en el repositorio por tamaño.

## Para añadir los sonidos:

Lee el archivo `SOUNDS_GUIDE.md` para instrucciones detalladas.

Archivos necesarios en esta carpeta:
- rain.mp3
- fire.mp3
- cafe.mp3
- ocean.mp3
- forest.mp3

Mientras tanto, seleccionar cualquier sonido no producirá audio hasta que añadas los archivos.
